2) ALUR STANDAR

DETECT → READ → BACKUP → (DMI/ME opsional) → ERASE+WRITE+VERIFY / INSTANT MODE → VERIFY LOLOS → rakit/tes nyala
  • Detect — catat chip + voltage
  • Read — hex bukan all-FF (kecuali memang kosong)
  • Backup — wajib, walau rusak. Unit mahal: 2–3× + MD5 sama
  • DMI Inject — pindah serial/key ke clean BIOS
  • ME Clean — late display / mati 30 menit
  • Instant Mode — paling aman untuk full reflash
